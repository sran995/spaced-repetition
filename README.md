# 今天，你复习了吗？

基于艾宾浩斯遗忘曲线的间隔复习系统。支持概念理解和精确记忆两种模式。

## 快速开始

1. 下载 `index.html`
2. 双击用浏览器打开
3. 创建用户 + 4 位 PIN → 进入

## 可选配置

进入右上角用户名 → **设置**：

- **AI API Key** — 可选。填入 DeepSeek 或 Claude API Key 后，英语模块自动翻译、概念理解 AI 判断。不填也能用：
  - 英语翻译改手动输入
  - 概念判断改为左右对照，自己对比

## 技术

- 纯 HTML/CSS/JS，零安装
- 数据存储在浏览器本地（localStorage）
- 不需要服务器、不需要数据库
- 部署于 GitHub Pages

## 导出/导入

设置页支持 JSON 格式导出和导入，用于备份或跨设备迁移。
